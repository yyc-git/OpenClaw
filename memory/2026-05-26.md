# 2026-05-26 对话记录

## QMD 记忆系统安装配置（07:00-07:18）

兄弟要求启动 **QMD（Quantum Memory Database）** 记忆系统，参考知乎文章：
https://zhuanlan.zhihu.com/p/2004224376657159734

### 安装步骤
1. **安装 bun** — `npm i -g bun`
2. **安装 QMD** — `bun install -g github:tobi/qmd`
3. **Build QMD** — 在 `~\.bun\install\global\node_modules\@tobilu\qmd` 下 `bun install && bun run build`
4. **安装 SQLite** — 下载 SQLite 工具到 `C:\sqlite`，加入 PATH
5. **配置 openclaw.json** — 添加 `memory.backend: "qmd"` （timeoutMs: 8000）
6. **下载 3 个 GGUF 模型** — 通过 `qmd pull`（需设置 `HF_ENDPOINT=https://hf-mirror.com` 解决国内访问 HuggingFace 的问题）
7. **重启 Gateway** — `openclaw gateway restart`

### 安装的模型（~2.1GB）
- embeddinggemma-300M-Q8_0（318MB）— 嵌入
- qwen3-reranker-0.6b-q8_0（610MB）— 重排序
- qmd-query-expansion-1.7B-q4_k_m（1.22GB）— 查询扩展

### 硬件环境
- GPU: NVIDIA RTX 2060 SUPER（8GB VRAM，6.9GB 空闲）
- Vulkan GPU 加速启用

### 验证结果
- `memory_search` provider 显示为 `qmd`
- search mode: BM25
- 响应时间: ~1.7s
- 正常工作中

### 注意
- 国内下载 HuggingFace 模型需设置 `HF_ENDPOINT=https://hf-mirror.com`
- PATH 需添加 `C:\Users\Administrator\.bun\bin` 和 `C:\sqlite`
