## E2E 场景3 — 性能录制（2026-06-16 15:21~15:30）
- **脚本**：`test/e2e/e2e-scenario3.cjs`
- **双窗口独立录制**：两个独立 Chrome 进程，各自有浮层按钮「▶ 录制此窗口」
  - 点哪个窗口的按钮 → CDP Profiler 只录制那个窗口
  - 支持 player1 或 player2 任选录制
- **自适应宽度**：检测屏幕宽度（1920px），平分窗口各 960px
- **数据采集**：FPS（rAF 挂钩）+ CPU Profile（CDP Profiler）+ API 耗时 + GameState 间隔
- **修复 FPS 时间戳**：用 FPS 数据第一条的 t 做基准（避免 Node vs 浏览器 performance.now() 时钟不一致）
- **结果示例**：player2 11.8s，平均 FPS 106，最低 29，AllCommands 419 次，性能良好 ✅
